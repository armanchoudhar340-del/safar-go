# TravelGo - Travel Booking Website

A responsive travel booking website template built with HTML5 and CSS3.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Booking Widget**: CSS-only tab system for switching between Flights, Trains, Buses, and Hotels.
- **Search Functionality**: Simulated search that passes parameters to a results page (`listings.html`).
- **Modern UI**: Clean interface with FontAwesome icons and Google Fonts (Poppins).

## Project Structure
- `index.html`: Main landing page with hero section, booking form, and popular routes.
- `listings.html`: Search results page with filters and mock listings.
- `style.css`: Main stylesheet containing all styles and responsive rules.
- `assets/`: Directory for project images (currently using Unsplash placeholders).

## technologies Used
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- FontAwesome (Icons)
- Google Fonts (Typography)

## Setup
Simply open `index.html` in your web browser to view the project.
